# Claude Manager
A cleanly designed tool for managing your Anthropic Claude Code session contexts, modular skills, and systemic memory.

## Setup
1. Clone this repository to your machine.
2. Run `Launch Claude Manager.bat` (Windows) to start the server on localhost:3847 and open the UI.
3. Configure your custom skills in the `skills/` folder and assign them in `server/server.js` and `ui/data.js` to manage their toggles.
